# Bioinformatics Demo for Google Colab

This repository contains a simple hands-on bioinformatics demo to be run in Google Colab.

## Repo Structure
- `data/` : Sample datasets (FASTA, gene expression CSV)
- `notebooks/` : Demo notebook for Colab
- `README.md` : Instructions

## How to Run in Google Colab
[Open in Colab](https://colab.research.google.com/github/<username>/bioinfo-demo/blob/main/notebooks/demo_colab.ipynb)

## Hands-on Tasks
1. Compute GC%, AT content, and reverse complements from FASTA sequences.
2. Visualize gene expression data as bar charts.
3. Upload your own FASTA/CSV files and repeat the analysis.
